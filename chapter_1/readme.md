# First Method
- We can run the rust program using rustc {filename_with_extension}
- Example -
  - --> rustc main.rs
- It produces a executable file , to run it --> ./main

# 2nd Method.
- Using cargo create a new directory.
- --> cargo new hello_cargo
- Enter that directory using 
  -   --> cd hello_cargo
- --> cargo build
- --> cargo run
